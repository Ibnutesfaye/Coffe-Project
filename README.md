# Coffee Paradise Website

A responsive, modern, and interactive website designed for "Coffee Paradise", a premium coffee shop. This project showcases the cafe's menu, gallery, testimonials, and provides a way for customers to contact the shop.

## Features

- **Responsive Design:** Fully responsive layout that adapts to all screen sizes (Desktop, Tablet, Mobile).
- **Interactive Navigation:**
  - Sticky header with smooth scrolling navigation.
  - Mobile hamburger menu for smaller screens.
  - Active section highlighting in the navigation bar.
- **Hero Section:** Eye-catching greeting with call-to-action buttons.
- **Menu Showcase:** Categorized display of offerings (Hot Beverages, Cold Beverages, Refreshments) with hover effects.
- **Testimonials Slider:** Interactive slider to display customer reviews with auto-play functionality.
- **Image Gallery:** a curated gallery of food and drink images.
- **Contact Form:** Functional contact form layout with client-side validation (Name, Email, Subject, Message).
- **Animations:** Smooth transitions, hover effects, and fade-in animations for a polished user experience.

## Technologies Used

- **HTML5:** Semantic markup for structure.
- **CSS3:** Custom styling, Flexbox, Grid, CSS Variables, and Media Queries for responsiveness.
- **JavaScript (Vanilla):** DOM manipulation, event handling, sliders, form validation, and scroll effects.
- **Google Fonts:** Usage of 'Poppins' font for typography.
- **SVG:** Scalable Vector Graphics for icons.

## Installation and Usage

Since this is a static website, no complex installation or backend server is required.

1.  **Clone the Repository (or download the files):**
    ```bash
    git clone <repository-url>
    ```
2.  **Open the Project:**
    Navigate to the project folder.
3.  **Run the Website:**
    Simply double-click the `index.html` file to open it in your default web browser.
    *Alternatively, you can use a live server extension (like Live Server in VS Code) for a better development experience.*

## How to Customize

-   **Content:** Edit the text within `index.html` to update menus, testimonials, or contact info.
-   **Styles:** Modify `styles.css` to change colors, fonts, or layout spacing. The primary color is currently set to `#FF9F1C` (Orange).
-   **Images:** Replace the image files in the root directory with your own, ensuring the filenames match those in `index.html` or update the `src` attributes in the HTML.
